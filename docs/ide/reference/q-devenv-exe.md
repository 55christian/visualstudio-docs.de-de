# <a name="-devenvexe"></a>/? („devenv.exe“)

Zeigt ein Nachrichtenfeld mit allen `devenv`-Schaltern mit einer kurzen Beschreibung jedes Schalters an.

## <a name="syntax"></a>Syntax

```shell
devenv /?
```
