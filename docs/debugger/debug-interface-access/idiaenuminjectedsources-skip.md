# <a name="idiaenuminjectedsourcesskip"></a>IDiaEnumInjectedSources::Skip
Überspringt eine angegebene Anzahl von injizierten Quellen in einer enumerationssequenz.

## <a name="syntax"></a>Syntax

```C++
HRESULT Skip ( 
   ULONG celt
);
```

#### <a name="parameters"></a>Parameter
 celt

in Die Anzahl der eingefügten Quellen in der zu über springenden enumerationssequenz.

## <a name="return-value"></a>Rückgabewert
 Wenn erfolgreich, wird `S_OK` zurückgegeben. Andernfalls wird `S_FALSE` zurückgegeben, wenn keine weiteren injizierten Quellen zum Überspringen vorhanden sind.
