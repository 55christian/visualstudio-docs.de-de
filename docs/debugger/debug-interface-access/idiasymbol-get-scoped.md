# <a name="idiasymbolget_scoped"></a>IDiaSymbol::get_scoped
Ruft ein Flag ab, das angibt, ob der benutzerdefinierte Datentyp in einem nicht globalen lexikalischen Gültigkeitsbereich angezeigt wird.

## <a name="syntax"></a>Syntax

```C++
HRESULT get_scoped ( 
   BOOL* pRetVal
);
```

#### <a name="parameters"></a>Parameter
 `pRetVal`

vorgenommen Gibt `TRUE` zurück, wenn der benutzerdefinierte Datentyp in einem nicht globalen lexikalischen Gültigkeitsbereich angezeigt wird. Andernfalls wird `FALSE` zurückgegeben.

## <a name="return-value"></a>Rückgabewert
 Wenn erfolgreich, wird `S_OK` zurückgegeben. Andernfalls wird `S_FALSE` oder ein Fehlercode zurückgegeben.

> [!NOTE]
> Der Rückgabewert `S_FALSE` bedeutet, dass die Eigenschaft für das Symbol nicht verfügbar ist.
