# <a name="idiastackframeget_size"></a>IDiaStackFrame::get_size
Ruft die Größe des Stapel Rahmens in Bytes ab.

## <a name="syntax"></a>Syntax

```C++
HRESULT get_size ( 
   DWORD* pRetVal
);
```

#### <a name="parameters"></a>Parameter
 `pRetVal`

vorgenommen Gibt die Größe des Stapel Rahmens in Bytes zurück.

## <a name="return-value"></a>Rückgabewert
 Gibt bei Erfolg `S_OK` zurück. Gibt `S_FALSE` zurück, wenn die Eigenschaft nicht unterstützt wird. Andernfalls wird ein Fehlercode zurückgegeben.
