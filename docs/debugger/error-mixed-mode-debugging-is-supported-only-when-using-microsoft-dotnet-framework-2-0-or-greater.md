# <a name="error-mixed-mode-debugging-is-supported-only-when-using-microsoft-net-framework-20-or-greater"></a>Fehler: Debuggen im gemischten Modus wird nur bei Verwendung von Microsoft .NET Framework, Version 2.0 oder höher, unterstützt
Um gemischten systemeigenen und verwalteten Code zu debuggen, müssen Sie .NET Framework, Version 2.0, 3.0 verfügen. 3.5 oder 4 verwenden. Im gemischten Modus mit früheren Versionen von .NET Framework-Debuggen wird nicht unterstützt.

### <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler

- Aktualisieren Sie .NET Framework, Version 2.0, 3.0, 3.5 oder 4.0.
