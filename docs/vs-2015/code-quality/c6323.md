# <a name="c6323"></a>C6323
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C6323: Verwendung eines arithmetischen Operators auf Boolean-Typ(en)  
  
 Diese Warnung tritt auf, wenn die arithmetische Operatoren auf Boolean-Datentypen verwendet werden. Verwendung von falscher Operator m√∂glicherweise falsche Ergebnisse liefern. Es gibt auch an, dass der Programmierer nicht im Code wiedergegeben werden.  
  
## <a name="example"></a>Beispiel  
 Der folgende Code generiert diese Warnung:  
  
```  
int test(bool a, bool b)  
{  
    int c = a + b;     //C6323  
    return c;  
}  
```  
  
 Um diese Warnung zu korrigieren, verwenden Sie richtigen Datentyp und dem Operator.  
  
```  
int test(int a, int b)  
{  
    int c = a + b;       
    return c;  
}  
```
