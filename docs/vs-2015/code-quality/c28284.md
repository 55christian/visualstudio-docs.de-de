# <a name="c28284"></a>C28284
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28284: Für das Symbol werden Prädikate derzeit nicht für nicht-funktionssymbole unterstützt  
  
 Diese Warnung gibt an, die einer bedingten Anmerkung (Prädikat `__drv_when`) auf einen anderen Wert als eine Funktion gefunden wurde.
