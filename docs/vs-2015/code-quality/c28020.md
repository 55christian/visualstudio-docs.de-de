# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28020: Der Ausdruck \<Expr > ist nicht der Fall bei diesem Aufruf  
  
 Diese Warnung wird ausgegeben bei der \_Satisfies\_ Ausdruck, der aufgeführt ist nicht "true". Häufig bedeutet dies einen falschen Parameter.  
  
 In diesem Fall auf die Deklaration einer Funktion geben Sie die Anmerkungen einer unmöglich Bedingung an.
