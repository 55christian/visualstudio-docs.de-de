[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28206: \<Ausdruck >: Linker Operand zeigt auf eine Struktur, verwenden `->`  
  
 Diese Warnung wird ausgegeben, wenn die Struktur zeigerdereferenzierungen Notation `->` wurde erwartet.
