# <a name="c26462-use_const_pointer_for_variable"></a>C26462 USE_CONST_POINTER_FOR_VARIABLE
  Der Wert, auf den von "% Variable%" gezeigt wird, wird nur einmal zugewiesen. Markieren Sie ihn als Zeiger auf `const`. Weitere Informationen finden [ C++ Sie unter Core Guidelines con. 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
