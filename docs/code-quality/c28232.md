# <a name="c28232"></a>C28232
Warnung C28232: \_Vor\_, \_Post\_, oder \_Deref\_ nicht auf keine Anmerkung angewendet.

 Diese Warnung gibt an, dass eine `_Pre_`, `_Post_`, oder `_Deref_` Operator in einem Annotation-Ausdruck, ohne eine nachfolgende funktionale Anmerkung angezeigt wird; der Modifizierer wurde ignoriert, aber dies weist darauf hin, einer falsch codierten Anmerkung.
