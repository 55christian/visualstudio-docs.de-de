# <a name="ca0056"></a>CA0056
Eine Projektdatei weist eine falsche Version des Analysetools auf.

Diese Warnung tritt auf, wenn FxCopCmd. exe manuell mit einer Projektdatei aufgerufen wird, die eine falsche Code Analysetool-Version aufweist.
