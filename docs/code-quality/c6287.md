# <a name="c6287"></a>C6287
Warnung C6287: Redundanter Code: Die Teilausdrücke links und rechts sind identisch.

 Diese Warnung gibt an, dass ein redundantes Element in einem Ausdruck erkannt wurde.

 Es ist schwierig, den Schweregrad dieses Problems zu beurteilen, ohne den Code zu untersuchen. Ein eigenständig vorhandener Test ist harmlos, aber die Konsequenzen beim Löschen des zweiten Tests können schwerwiegend sein. Der Code sollte überprüft werden, um sicherzustellen, dass ein Test nicht ausgelassen wurde.

## <a name="example"></a>Beispiel
 Der folgende Code generiert diese Warnung:

```
void f(int x)
{
  if ((x == 1) && (x == 1))
  {
    //logic
  }
  if ((x != 1) || (x != 1))
  {
    //logic
  }
}
```

 Der folgende Code zeigt verschiedene Möglichkeiten, diese Warnung zu beheben:

```
void f(int x, int y)
{
  /* Remove the redundant sub-expression: */
  if (x == 1)
  {
     // logic
  }
  if (x != 1)
  {
    // logic
  }
  /* or test the missing variable: */
  if ((x == 1) && (y == 1))
  {
     // logic
  }
  if ((x != 1) || (y != 1))
  {
     // logic
  }
}
```
