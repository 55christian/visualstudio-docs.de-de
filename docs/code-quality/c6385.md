# <a name="c6385"></a>C6385
Warnung C6385: Ungültige Daten: beim Zugriff auf \<buffer Name > ist die lesbare Größe \<size1 > bytes, aber \<size2 > Bytes werden möglicherweise gelesen: Zeilen: x, y

 Diese Warnung gibt an, dass der lesbare Wertebereich des angegebenen Puffers kleiner sein kann als der Index, der zum Lesen verwendet wird. Versuche, Daten außerhalb des gültigen Bereichs zu lesen, führen zu einem Pufferüberlauf.

## <a name="example"></a>Beispiel
 Der folgende Code generiert diese Warnung:

```
void f(int i)
{
   char a[20];
   char j;
   if (i <= 20)
   {
      j = a[i];
   }
}
```

 So korrigieren Sie die Warnung unter Verwendung des folgenden Codes

```
void f(int i)
{
   char a[20];
   char j;
   if (i < 20)
   {
      j = a[i];
   }
}
```
