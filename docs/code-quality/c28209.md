# <a name="c28209"></a>C28209
Warnung C28209: Die Deklaration für Symbol weist einen Konflikt

 Diese Warnung gibt an, die Deklaration eines nicht ordnungsgemäß erstellten Anmerkung. Diese Warnung sollte niemals während normaler Verwendung auftreten.
