# <a name="c28204"></a>C28204
Warnung C28204: \<function >: Nur eine dieser über Ladungen und die bei \<filename > (\<-Zeilen >) werden für \<paramname-> kommentiert: beide oder keine von beiden müssen mit Anmerkungen versehen werden.

 Diese Warnung wird gemeldet, wenn ein Fehler in den Anmerkungen vorliegt.
