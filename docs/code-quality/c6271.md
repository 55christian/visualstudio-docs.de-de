# <a name="c6271"></a>C6271
Warning C6271: zus채tzliches Argument, das an \<Funktion 체bergeben wird >: Parameter \<Number > wird von der Format Zeichenfolge nicht verwendet.

 Diese Warnung gibt an, dass zus채tzliche Argumente 체ber die von der Format Zeichenfolge festgelegten hinaus bereitgestellt werden. Dieser Fehler ist allein nicht sichtbar, obwohl er angibt, dass die Absicht des Programmierers nicht im Code widergespiegelt wird.

## <a name="example"></a>Beispiel
 Im folgenden Beispielcode wird diese Warnung generiert:

```cpp
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf(buff,"%d",1,2);
}
```

 So korrigieren Sie die Warnung unter Verwendung des folgenden Beispielcodes

```cpp
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf(buff,"%d, %d",1,2);
}
```

 Der folgende Beispielcode ruft die sichere Zeichen folgen Bearbeitungsfunktion `sprintf_s`auf, um diese Warnung zu beheben:

```cpp
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf_s( buff, 5,"%s %d", 1,2 ); //safe version
}
```
