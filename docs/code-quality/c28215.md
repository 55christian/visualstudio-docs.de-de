# <a name="c28215"></a>C28215
Warnung C28215: Der Typefix kann nicht auf einen Parameter angewendet werden, der bereits über einen Typefix verfügt.

 Das Anwenden einer `__typefix`-Anmerkung auf einen Parameter, der bereits über diese Anmerkung verfügt, ist ein Fehler. Die Anmerkungen `__typefix` werden nur in einigen besonderen Fällen verwendet, und es wird davon ausgegangen, dass diese Warnung nicht in normaler Verwendung zu sehen ist.
