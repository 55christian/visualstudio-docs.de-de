# <a name="c28284"></a>C28284
Warnung C28284: Für das Symbol werden Prädikate für nicht-Funktions Symbole zurzeit nicht unterstützt.

 Diese Warnung weist darauf hin, dass eine bedingte Anmerkung (Prädikat, `__drv_when`) in einem anderen als einer Funktion gefunden wurde.
