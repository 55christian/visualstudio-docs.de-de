# <a name="ca0501"></a>CA0501
Kann nicht gelesen werden Ausgabebericht

 Dies tritt auf, wenn es sich bei Ausgabebericht kann nicht geschrieben werden, kann eine der folgenden Gründe:

- Der Datenträger ist voll

- Keine Berechtigungen zum Schreiben in Verzeichnis

- Verzeichnis ist nicht vorhanden.
