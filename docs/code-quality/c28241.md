# <a name="c28241"></a>C28241
Warnung C28241: Die Anmerkung für die Funktion zum Parameter wird nicht erkannt.

 Es wurde eine unbekannte anmerkungsnamen verwendet. Dies sollte nicht möglich, wenn die standard-Makros für Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, normalen datenbanknutzung angesehen werden.
