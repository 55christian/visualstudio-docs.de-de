# <a name="iactivescriptauthorprocedure-interface"></a>IActiveScriptAuthorProcedure-Schnittstelle
Zusätzlich zu den Methoden, die von `IUnknown` geerbt werden, macht die `IScriptAuthorProcedure`-Schnittstelle die folgende Methode verfügbar.  
  
## <a name="methods-in-vtable-order"></a>Methoden in Vtable-Reihenfolge  
  
|Methode|Beschreibung|  
|------------|-----------------|  
|[IActiveScriptAuthorProcedure::ParseProcedureText](../../winscript/reference/iactivescriptauthorprocedure-parseproceduretext.md)|Analysiert eine Code Prozedur, fügt der Skript Erstellungs-Engine den Text der Code Prozedur hinzu und erstellt ein `IScriptEntry`-Objekt, das der Code Prozedur entspricht.|
