# <a name="iactivescriptauthoriscommitchar"></a>IActiveScriptAuthor::IsCommitChar
Gibt einen Wert zurück, der angibt, ob ein bestimmtes Zeichen einen Anweisungs Abschluss Commit durch die Anwendung auslöst.  
  
## <a name="syntax"></a>Syntax  
  
```cpp
HRESULT IsCommitChar(  
   OLECHAR    ch,  
   BOOL       *pfcommit  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `ch`  
 in Das zu überprüfende Zeichen.  
  
 `pfcommit`  
 [out] `True`, wenn das Zeichen ein Commit-Zeichen ist. Andernfalls `False`.  
  
## <a name="return-value"></a>Rückgabewert  
 Eine `HRESULT`. Mögliches Werte (aber nicht die Einzigen) sind die in der folgenden Tabelle.  
  
|Wert|Beschreibung|  
|-----------|-----------------|  
|`S_OK`|Die Methode war erfolgreich.|
