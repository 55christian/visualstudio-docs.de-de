# <a name="script_e_reported-error-code"></a>SCRIPT_E_REPORTED-Fehlercode
Die Skript-Engine hat über [iactivescriptsite:: onscripterror](../../winscript/reference/iactivescriptsite-onscripterror.md)eine nicht behandelte Ausnahme an den Host gemeldet. Dieser Fehler kann vom Host ignoriert werden.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
SCRIPT_E_REPORTED   0x80020101L  
```
