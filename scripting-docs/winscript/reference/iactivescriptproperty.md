# <a name="iactivescriptproperty"></a>IActiveScriptProperty
Die `IActiveScriptProperty`-Schnittstelle wird verwendet, um Konfigurations Eigenschaften zu erhalten und festzulegen.  
  
## <a name="methods-in-vtable-order"></a>Methoden in Vtable-Reihenfolge  
  
|Methode|Beschreibung|  
|------------|-----------------|  
|[GetProperty](../../winscript/reference/iactivescriptproperty-getproperty.md)|Ruft die Eigenschaft ab, die durch den-Parameter angegeben wird.|  
|[SetProperty](../../winscript/reference/iactivescriptproperty-setproperty.md)|Legt die-Eigenschaft fest, die durch den-Parameter angegeben wird.|
