# <a name="iremotedebugapplicationenumthreads"></a>IRemoteDebugApplication::EnumThreads
Listet alle Threads auf, die bekanntermaßen der Anwendung zugeordnet sind.  
  
## <a name="syntax"></a>Syntax  
  
```cpp
HRESULT EnumThreads(  
   IEnumRemoteDebugApplicationThreads**  pperdat  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `pperdat`  
 vorgenommen Enumerator, der alle Threads auflistet, die bekanntermaßen der Anwendung zugeordnet sind.  
  
## <a name="return-value"></a>Rückgabewert  
 Die Methode gibt ein `HRESULT` zurück. Mögliches Werte (aber nicht die Einzigen) sind die in der folgenden Tabelle.  
  
|Wert|Beschreibung|  
|-----------|-----------------|  
|`S_OK`|Die Methode war erfolgreich.|  
  
## <a name="remarks"></a>Hinweise  
 Diese Methode listet alle Threads auf, die bekanntermaßen der Anwendung zugeordnet sind. Neue Threads können jederzeit hinzugefügt werden.
