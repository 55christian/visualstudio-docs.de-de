# <a name="undefined-identifier"></a>Nicht definierter Bezeichner
Sie haben versucht, einen Bezeichner zu verwenden, der vom [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] Compiler nicht erkannt wird. Der nicht definierte Wert wird zur√ºckgegeben, wenn Sie verwenden:  
  
- eine Variable, die nicht vorhanden ist.

- eine Objekt Eigenschaft, die nicht vorhanden ist.
